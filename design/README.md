# Nicoletti Premium Cigar — Design Assets

This folder hosts static design assets for Nicoletti Premium Cigar:

- HTML / CSS mockups
- Brand guidelines and visual references
- Logo files and brand imagery
- Designer-provided artifacts before they are translated to components

## GitHub Pages

Once content is added here, GitHub Pages can be enabled in the repository
settings to serve this folder, making the design assets accessible at:

```
https://amandaalmeidda.github.io/client-website-nicoletti-premium-cigar/
```

To enable:

1. Go to the repository's **Settings -> Pages**
2. Source: **Deploy from a branch**
3. Branch: **main**, Folder: **/design**

> Note: The repository is created with GitHub Pages ready to be configured.
> Enable it once initial design assets land in this folder.

## Folder conventions

```
design/
├── index.html          # Entry point for the design preview
├── styles/             # CSS files
├── assets/             # Images, logos, fonts
└── mockups/            # Standalone HTML mockups
```
